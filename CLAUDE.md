## Project Structure

- `docs/PEP-001-infra.md` — full architecture document

## Deploy

Server is ARM64, local machine is x86_64. Can't cross-build, so the server builds from the GitHub flake:

```bash
ssh root@46.225.161.186 "nixos-rebuild switch --flake github:denisraison/infra#prod"
```

Update rekan to a specific release or latest, then push and deploy:

```bash
nix flake lock --override-input rekan github:denisraison/rekan/v0.2.0  # specific release
nix flake lock --update-input rekan                                     # latest main
git add flake.lock && git commit -m "Update rekan" && git push
ssh root@46.225.161.186 "nixos-rebuild switch --flake github:denisraison/infra#prod"
```

## OpenTofu

Run from `tofu/` directory. Secrets are in `terraform.tfvars` (gitignored).

```bash
nix run nixpkgs#opentofu -- plan
nix run nixpkgs#opentofu -- apply
```

## Adding an App

Each app repo exports a `nixosModules.default` from its flake. To add one:

1. Add the app as a flake input in `flake.nix`
2. Import its module in `nixosConfigurations.prod.modules`
3. Configure `services.<app>` in `nixos/configuration.nix`
4. Deploy

## Important

- `nixos/networking.nix` and `nixos/hardware-configuration.nix` were generated by nixos-infect. Do not edit unless the server is reprovisioned.
- `terraform.tfvars` contains the Hetzner API token. Never commit it.
- `system.stateVersion` in configuration.nix must stay at "23.11" (set by initial install).
